#  Arabic NLP Dataset Analysis

This project focuses on analyzing Arabic text datasets for two main NLP tasks:
- **Sarcasm Detection**
- **Sentiment Analysis**

---

## Data Folder

The `data/` folder contains all Sample ,Source of Dataset and scripts decribe it to understand for this project.

### Contents:

- **ArSarcasm_sample.csv**  
  Sample dataset for Arabic sarcasm detection.  
  Includes tweet text, sarcasm label, sentiment, and dialect.

- **Sentiment_tweets_sample.txt**  
  Sample dataset for Arabic sentiment analysis (ASTD).  
  Each line contains a sentence and its sentiment label (POS, NEG, OBJ).

- **script_data.py**  
  Python script used to:
  - Load datasets  
  - Display dataset information  
  - Show sample data  

- **README.md (inside data folder)**  
  Provides detailed description of dataset structure and fields.
