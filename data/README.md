#Data Folder - Arabic NLP Project

This folder contains sample datasets used for Arabic text analysis, including sarcasm detection and sentiment analysis.

---

## Datasets Overview

### 1. ArSarcasm_sample.csv
- **Description:**  
  Sample dataset for Arabic sarcasm detection.
- **Task:**  
  Classify whether a tweet is sarcastic or not.
- **Columns:**
  - `tweet` → Arabic text
  - `sarcasm` → True / False
  - `sentiment` → sentiment label
  - `dialect` → Arabic dialect (e.g., msa, gulf)
- **Source:**  
  https://github.com/iabufarha/ArSarcasm


### 2. Sentiment_tweets_sample.txt
- **Description:**  
  Sample dataset for Arabic sentiment analysis.
- **Task:**  
  Classify sentiment of tweets.
- **Format:**  
  Each line contains:
  - **Labels:**
- `POS` → Positive sentiment  
- `NEG` → Negative sentiment  
- `OBJ` → Objective / Neutral sentiment  
- **Source:**  
https://github.com/mahmoudnabil/ASTD
